TRABAJANDO CON RAMAS
    git checkout 
        cambiar rama

    git checkout -b "nombre"
        cambiar a la nueva rama

    git branch -D /nombreBranch

SUBIR CAMBIOS
    git add -A 
        agrega cambios a la transacción

    git commit -m "mensaje descriptivo en imperativo"

    git push _ origin /nombreBranch _

    TRABAJANDO CON LA VERSIÓN MÁS RECIENTE: git pull

    JUNTAR RAMAS: te mueves a la que le quieres añadir
        (normalmente main) git merge /nombreBranch
